# Angel's Art Website
A personal art portfolio website built with HTML, CSS, and JavaScript to showcase some of Angel's artwork, animations, and creative journey.

View the website here: https://angelartpage.netlify.app 

## Project Structure

```text
├── images/
├── videos/
├── .gitgnore
├── index.html
├── style.css
├── script.js
└── README.md
```

## Features
- Responsive design (cross-platform with desktop and mobile screens)
- Interactive artwork gallery with clickable zoom in
- Embedded animations and videos
- Social media links for easy navigation
